What changed vs. the classic version

Classic (v1) — One-shot LLM calls

  • Web form to enter research brief
  • Single Gemini call generates all questions
  • No quality review of questions
  • State machine decides when to probe
  • Single Gemini call generates full report
  • No self-critique of analysis

Agentic (v2) — Tool-calling loops

  • BriefAgent chats with client to collect brief
  • DesignerAgent generates then self-reviews questions
  • Iterative revision until quality bar is met
  • InterviewAgent decides to probe based on answer quality
  • AnalysisAgent runs 4 passes: extract → synthesize → write → critique
  • Self-critique patches weak sections before finalising
